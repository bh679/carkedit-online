Three ways to add “How to Play” — no scrolling

The lobby was tipping into a scroll once the instructions entry point was added. Each option keeps every control on one screen: same room code, players, settings and the Ready / Start / Leave buttons — they differ only in where How-to-Play lives and how it’s emphasised for a first-time host.
